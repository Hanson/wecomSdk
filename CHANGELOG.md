# Changelog

## 2025-12-17 — v0.0.1
- 初始公开版本，模块名 `github.com/Hanson/wecomSdk`
- 统一调用方式：`GetJSON` / `PostJSON` / `CallJSON` / `GetJSONWithReq`
- 错误统一：始终填充响应结构；`errcode!=0` 返回 `wecom.Error{Code, Message, Raw}`
- 结构化类型：按接口路径命名（成员、部门、标签、客户联系、微信客服、审批、考勤、日历/日程、应用菜单、素材）
- 微信客服：账号列表、发送消息、同步消息、会话状态、联系方式
- 客户联系：备注、跟进人列表、企业客户标签增改删查
- 通讯录：批量删除、ID 转换（openid ↔ userid）
- OA 审批：审批详情、模板详情
- 考勤：打卡数据、日汇总
- 日历/日程：新增/查询/更新/删除
- 应用菜单：创建/删除/查询
- 素材：新增 `GetRaw` 支持二进制下载
- 回调：URL 验证与消息接收（AES-CBC + PKCS7，签名校验）
- 测试：本地模拟与真实集成测试（读取环境变量）；示例用户脱敏为 `demo_user`
- 文档：README 安装、使用示例与回调指引
